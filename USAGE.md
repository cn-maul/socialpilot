# SocialPilot 使用指南

## 快速开始

### 1. 启动服务

```bash
./socialpilot web
# 或自定义端口
./socialpilot web --port 3000
# 允许外部访问
./socialpilot web --host 0.0.0.0 --port 8080
```

### 2. 配置 AI（重要！）

首次使用时，以下功能需要 AI 配置才能使用：
- ✨ 更新人物画像
- 💬 智能回复建议
- 📝 结构化录入（需要 LLM 解析时）

**配置步骤：**
1. 打开网页 `http://127.0.0.1:8080`
2. 点击左侧菜单「设置」
3. 填写以下信息：
   - **Base URL**: LLM API 地址（例如：`https://api.openai.com/v1`）
   - **API Key**: API 密钥
   - **Model**: 模型名称（例如：`gpt-4`）
   - **Timeout**: 请求超时时间（默认 60 秒）
4. 点击「保存设置」

配置完成后，页面会自动识别并启用 AI 相关功能。

### 3. 添加联系人

1. 在「联系人与详情」页面
2. 在「新增联系人」卡片中填写：
   - 姓名（必填）
   - 性别
   - 标签（同事/同学/朋友/亲戚）
3. 点击「创建联系人」

### 4. 使用 AI 功能

配置 AI 后，你可以使用：

**📊 更新人物画像**
- 点击联系人卡片右上角的 ✨ 图标
- AI 会分析历史对话，生成人物画像

**💬 智能回复建议**
- 在「对方新消息」输入对方的消息
- 点击「生成建议」
- AI 会提供多种回复建议
- 选择一个建议后可编辑，然后「采纳并回写」

**📝 结构化录入**
- 输入对话描述或 JSON 格式的聊天记录
- AI 会自动解析并结构化存储

## 文件位置

所有数据文件都在二进制文件同目录下：

- `config.json` - 配置文件（首次保存设置时创建）
- `socialpilot.db` - SQLite 数据库（首次添加联系人时创建）

## 常见问题

### Q: 提示"AI 配置缺失"怎么办？
**A:** 请先在「设置」页面配置 Base URL、API Key 和 Model。

### Q: 配置文件在哪里？
**A:** 配置文件在二进制文件同目录下的 `config.json`，可直接编辑。

### Q: 数据库可以迁移吗？
**A:** 可以，直接复制 `socialpilot.db` 文件即可。

### Q: 支持哪些 LLM？
**A:** 任何兼容 OpenAI API 格式的 LLM 都支持。

```bash
./build.sh

cd webui && npm run build && cd ..
go build -o socialpilot
```

## 高级设置

### 自定义 AI 提示词

在「设置」页面可以自定义以下提示词：
- **Extract**: 结构化录入时的提取提示词
- **Copilot**: 智能回复建议的提示词
- **Analyze**: 人物画像分析的提示词
- **Compress**: 会话压缩的提示词

留空则使用默认提示词。
